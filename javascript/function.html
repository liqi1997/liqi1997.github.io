JavaScript 函数

我认为函数的意义就是封装以及封装带来的额外好处，比如要开发一款计算器程序，该程序第一个功能是求两个数字之间的和，程序员写几个语句，接收用户输入的两个数字和加法符号，相加后将新的数字返回给用户， 功能开发完毕，这时候想要扩展该程序的功能，能够实现减法，这时候再写几个语句也实现了功能，随着功能越来越多，程序代码越写越多，而语句的组合方式只能一行一行向下，这样的程序没有办法维护。

这时候如果针对不同的功能编写不同的函数，放置在不同的文件中，公共逻辑是获取用户输入的值和符号，根据不同的符号执行不同的函数，就实现了剥离。 而且像加法函数和减法函数由于功能类似，可以开发通用性强一点的函数来实现此功能，这就降低了代码冗余度。
