JavaScript 解构

先看看 ES5 怎么从数组中取值

        var arr = [1, 2, { name: 'liqi' }];
        var element1 = arr[0]
        arr[0] = 100;
        console.log(element1)
        var element3 = arr[2]
        arr[2].name = 'lq'
        console.log(element3);

下面介绍解构赋值的作用：传统写法的简写

        var [e1, e2, e3] = arr;
        arr[1] = 200;
        arr[2].name = 'LI'
        console.log(e1, e2, e3)

解构赋值的工作原理：模式匹配，等号两侧模式相同即解构成功

        var [age1, [age2]] = [100, [200, 300]];
        console.log('age', age1, age2)
    
    var [age1, , age3] = [1, 2, 3, 4];
    console.log(age1, age3)
    var [element1] = []
    console.log(element1)

    var [e1, ...others] = [1, 2, 3, 4]
    console.log(others, typeof others, others instanceof Array)

等号右边不是可迭代结构会报错，所以说解构的原理就是通过迭代器对象来查找并赋值
